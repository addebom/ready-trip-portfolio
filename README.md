# 준비해여행

### 여행 준비 과정에서 반복적으로 발생하는 준비물 누락 문제를 해결하는 체크리스트 서비스

준비해여행은 여행 전 준비물을 놓치지 않도록 도와주는 여행 준비 체크리스트 서비스입니다.

사용자가 여행 전 챙겨야 할 항목을 정리하고, 준비 상태를 확인할 수 있도록 설계했습니다.

---
</div>
## 프로젝트 개요

여행을 준비할 때 많은 사용자는 비슷한 문제를 반복해서 겪습니다.

* 무엇을 챙겨야 할지 매번 다시 생각해야 함
* 준비물을 빠뜨릴까 불안함
* 상황별 준비물이 머릿속에서 정리되지 않음
* 여행 직전 체크가 번거로움

준비해여행은 이 문제를 체크리스트 기반 서비스로 구조화하여 해결하고자 했습니다.

---

## 핵심 목표

* 여행 준비물을 한눈에 확인할 수 있게 만들기
* 사용자가 준비 상태를 쉽게 체크할 수 있게 만들기
* 여행 전 불안감을 줄이는 간단한 서비스 흐름 만들기
* 실제 사용자가 접근 가능한 형태로 출시하기

* 준비물 완료 여부 체크
* 여행 전 준비 상태 확인
* 앱인토스 환경에서 실행 가능한 서비스 구성

## 문제 해결 과정

### 1. 문제 발견

여행 준비는 반복적인 일이지만, 사용자는 매번 준비물을 다시 떠올려야 합니다.

이 과정에서 준비물 누락, 체크 피로, 여행 직전 불안감이 발생한다고 판단했습니다.

### 2. 서비스 구조화

복잡한 기능보다 사용자가 바로 이해할 수 있는 체크리스트 흐름을 우선했습니다.

처음부터 많은 기능을 넣기보다, 여행 준비라는 핵심 문제를 단순한 사용 흐름으로 해결하는 데 집중했습니다.

### 3. 출시 및 운영

학습용 프로젝트에 그치지 않고 실제 사용자가 접근할 수 있는 형태로 출시했습니다.

출시 이후 약 1개월 동안 누적 사용자 약 110명이 이용했으며, 일 사용자 1명에서 7명 수준의 이용이 발생했습니다.

이 프로젝트를 통해 서비스는 개발로 끝나는 것이 아니라는 점을 배웠습니다.

실제 사용자가 들어오면 다음과 같은 질문이 생깁니다.

* 사용자는 어떤 경로로 들어오는가
* 어떤 기능을 실제로 사용하는가
* 처음 기대한 흐름대로 행동하는가
* 개선해야 할 지점은 무엇인가

준비해여행은 작은 서비스이지만, 문제 정의부터 출시, 운영, 사용자 관찰까지 경험한 프로젝트입니다.
