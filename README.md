## 프로젝트 소개

여행을 준비할 때 사람들은 반복적으로 비슷한 문제를 경험합니다.

* 무엇을 챙겨야 할지 매번 다시 생각해야 함
* 준비물을 빠뜨릴까 불안함
* 출국 준비와 귀국 준비가 섞여 있음
* 여행 상황에 따라 필요한 준비물이 달라짐

준비해여행은 이러한 문제를 해결하기 위해 제작한 여행 준비 체크리스트 서비스입니다.

단순한 체크리스트를 넘어 여행 상황에 따라 필요한 준비물을 추천하고 관리할 수 있도록 설계했습니다.

## 문제 해결 과정

### 문제 발견

여행 준비 과정에서 준비물을 빠뜨리는 일이 반복적으로 발생했습니다.

### 해결 방법

사용자의 여행 상황에 맞는 준비물을 추천하고 체크할 수 있는 서비스 흐름을 설계했습니다.

### 결과

실제 서비스를 출시하고 사용자 이용 데이터를 관찰하며 운영 경험을 쌓을 수 있었습니다.

## 배운 점

이 프로젝트를 통해 서비스는 개발로 끝나는 것이 아니라는 점을 배웠습니다.

실제 사용자가 생기면 다음과 같은 질문이 생깁니다.

* 사용자는 어떤 기능을 사용하는가?
* 어디에서 이탈하는가?
* 어떤 기능을 더 원할까?
* 어떻게 개선해야 할까?

준비해여행은 작은 프로젝트였지만

문제 정의 → 기획 → 개발 → 출시 → 운영

전 과정을 경험할 수 있었던 프로젝트입니다.
