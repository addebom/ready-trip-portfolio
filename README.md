# 준비해여행

### 여행 상황에 맞는 준비물을 추천하고 관리할 수 있는 여행 준비 체크리스트 서비스

## Overview

준비해여행은 토스 미니앱(Apps in Toss) 환경에서 운영 중인 여행 준비 체크리스트 서비스입니다.

여행 준비 과정에서 반복적으로 발생하는 준비물 누락 문제를 해결하기 위해 개발했습니다.
