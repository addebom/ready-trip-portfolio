# 준비해여행

여행 상황에 맞는 준비물을 추천하고 관리할 수 있는 여행 준비 체크리스트 서비스

---

## Overview

준비해여행은 여행 준비 과정에서 반복적으로 발생하는 준비물 누락 문제를 해결하기 위해 개발한 서비스입니다.

사용자는 출국 준비와 귀국 준비를 분리하여 관리할 수 있으며, 여행 상황에 맞는 추천 준비물을 확인할 수 있습니다.

---

## Why

여행을 준비할 때 사람들은 반복적으로 비슷한 문제를 경험합니다.

* 무엇을 챙겨야 할지 매번 다시 생각해야 함
* 준비물을 빠뜨릴까 불안함
* 출국 준비와 귀국 준비가 섞여 있음

준비해여행은 이러한 문제를 체크리스트 기반 서비스로 해결하기 위해 시작되었습니다.
