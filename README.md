# 준비해여행

### 여행 상황에 맞는 준비물을 추천하고 관리할 수 있는 여행 준비 체크리스트 서비스

<p align="center">
실제 서비스 출시 · 현재 운영 중
</p>

---

## Overview

준비해여행은 여행 준비 과정에서 반복적으로 발생하는 준비물 누락 문제를 해결하기 위해 개발한 서비스입니다.

사용자는 출국 준비와 귀국 준비를 분리하여 관리할 수 있으며, 여행 상황에 맞는 추천 준비물을 확인할 수 있습니다.

단순한 체크리스트를 넘어 여행 준비 과정을 구조화하고, 반복적으로 발생하는 불편함을 줄이는 것을 목표로 했습니다.

---

## Why

여행을 준비할 때 사람들은 반복적으로 비슷한 문제를 경험합니다.

* 무엇을 챙겨야 할지 매번 다시 생각해야 한다.
* 준비물을 빠뜨릴까 불안하다.
* 출국 준비와 귀국 준비가 섞여 있다.
* 여행 상황에 따라 필요한 준비물이 달라진다.

기존에는 메모장이나 메시지 앱을 이용해 준비물을 관리하는 경우가 많았지만, 반복 사용성과 관리 측면에서 한계가 있다고 판단했습니다.

준비해여행은 이러한 문제를 체크리스트 기반 서비스로 해결하기 위해 시작되었습니다.

## What I Learned

이 프로젝트를 통해 서비스는 개발로 끝나는 것이 아니라는 점을 배웠습니다.

실제 사용자가 생기면

* 어떤 기능을 사용하는지
* 어디에서 이탈하는지
* 어떤 기능을 원하는지
* 무엇을 개선해야 하는지

를 계속 관찰해야 한다는 점을 경험했습니다.

작은 규모의 서비스였지만

문제 정의 → 기획 → 개발 → 출시 → 운영

전 과정을 경험할 수 있었던 프로젝트였습니다.
