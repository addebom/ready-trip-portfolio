# Project Note - 준비해여행

## 프로젝트 소개

준비해여행은 여행 준비 과정에서 반복적으로 발생하는 준비물 누락 문제를 해결하기 위해 개발한 여행 준비 체크리스트 서비스입니다.

Apps in Toss(토스 미니앱) 환경에서 실제 서비스로 출시하여 운영하고 있습니다.

---

## 문제

여행을 준비할 때 사람들은 비슷한 문제를 반복적으로 경험합니다.

* 무엇을 챙겨야 하는지 매번 다시 생각해야 한다.
* 준비물을 빠뜨리는 경우가 발생한다.
* 출국 준비와 귀국 준비가 섞여 관리가 어렵다.
* 여행 상황에 따라 필요한 준비물이 달라진다.

---

## 해결 방법

준비해여행은 여행 준비 과정을 체크리스트 기반 서비스로 구조화했습니다.

주요 기능은 다음과 같습니다.

* 여행 준비 체크리스트
* 출국 / 귀국 분리 관리
* 상황별 추천 준비물 제공
* 중복 입력 방지
* 모바일 환경 최적화
